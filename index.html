It's made purely in HTML

HTML stands for Hyper Text Markup Language, and it isn't necessarily a programming language.

HTML does not run logic like other langauges do, such as Python or C++. It just reads from a .html file and lays out the page according to those instructions!
It shares similarities to compiler languages such as C++, where in the .html file spacing doesn't matter when regarding the final result.
The instructions in the file are organized as such to enable readability for those editing, just like C++!

The file can be stored locally and accessed without internet, which is why when right-clicking on a web page you have the option to save the page!
This style of page layout and simplicity gives nostalgia of the internet back in the early 2000's; but can still be seen today when things go wrong when visiting a website.
Sometimes when going to csusb.edu, it may not load properly and you will be presented with a disordered front page that appears to be only HTML.
NOTE: Notice that the link is either blue or purple. Blue means you haven't visited the website before (according to your history), and purple means that you have visited the website before.
NOTE: Also notice if you clicked on the link, it opened in another tab in the browser instead of the current tab in which you clicked the link? That's because the target attribute was set to _blank
